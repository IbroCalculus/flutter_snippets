========== 1. To use custom font ===========
- Right-click on project name and create new folder called ie 'assets/fonts'
- Copy and paste the fonts inside the folder, ie: Lettertype-Bold.ttf, Lettertype-Bold-italic.ttf, Lettertype-italic.ttf, Lettertype-Regular.ttf, Lettertype-SemiBold.ttf, Lettertype-SemiBold-italic.ttf

- Go to 'pubspec.yaml' file, scroll down to fonts section and uncomment all the codes

-NB: Identation matters, fonts in directly under # example, if fonts is moved 1 space back, others like family must move same
  # example:
  fonts:
     - family: Lettertype
       fonts:
        - asset: assets/fonts/Lettertype-Regular.ttf
          weight: 400

        - asset: assets/fonts/Lettertype-Italic.ttf
          weight: 400
          style: italic

        - asset: assets/fonts/Lettertype-Bold.ttf
          weight: 700

        - asset: assets/fonts/Lettertype-Bold-Italic.ttf
          weight: 700
          style: italic

        - asset: assets/fonts/Lettertype-SemiBold.ttf
          weight: 600

        - asset: assets/fonts/Lettertype-SemiBold-Italic.ttf
          weight: 600
          style: italic


------ in main.dart file -----------

        child: Column(
          children: [
            Text("Text 1", style: TextStyle(fontFamily: 'Lettertype', fontStyle: FontStyle.normal, fontWeight: FontWeight.bold, fontSize: 36)), // Bold
            Text("Text 2", style: TextStyle(fontFamily: 'Lettertype', fontStyle: FontStyle.italic, fontWeight: FontWeight.bold, fontSize: 36)), // Bold Italic
            Text("Text 3", style: TextStyle(fontFamily: 'Lettertype', fontStyle: FontStyle.italic, fontWeight: FontWeight.normal, fontSize: 36)), // Regular Italic
            Text("Text 4", style: TextStyle(fontFamily: 'Lettertype', fontStyle: FontStyle.normal, fontWeight: FontWeight.normal, fontSize: 36)), // Regular
            Text("Text 5", style: TextStyle(fontFamily: 'Lettertype', fontStyle: FontStyle.normal, fontWeight: FontWeight.w600, fontSize: 36)), // SemiBold
            Text("Text 6", style: TextStyle(fontFamily: 'Lettertype', fontStyle: FontStyle.italic, fontWeight: FontWeight.w600, fontSize: 36)), // SemiBold Italic
          ],
        ),
