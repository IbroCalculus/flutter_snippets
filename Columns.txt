NB: IntrinsicWidth: Wrapping a Column with this widget makes it take the width of its tallest child.
    It gives all the row children same width, by stretching them to match the width of the maximum.

    mainAxisAlignment: MainAxisAlignment.stretch, is important for that.

    ---------------- Example --------------------

      IntrinsicWidth(
      child: Column(
        mainAxisAlignment: MainAxisAlignment.stretch,
        children: [
          Expanded(
            child: Container(
              color: Colors.red,
              width: 50, // This child is short
              child: const Center(child: Text("Short", style: TextStyle(color: Colors.white))),
            ),
          ),
          Expanded(
            child: Container(
              color: Colors.green,
              width: 100, // This child is tall
              child: const Center(child: Text("Tall", style: TextStyle(color: Colors.white))),
            ),
          ),
          Expanded(
            child: Container(
              color: Colors.blue,
              width: 30, // This child is shorter
              child: const Center(child: Text("Short", style: TextStyle(color: Colors.white))),
            ),
          ),
        ],
      ),
    ),

                onPressed: () {

                  );
                },
