dependencies:
  rive: ^0.14.7

------------------- Setup ----------------
Go to: rive.app (https://editor.rive.app/home) OR Use the desktop app


------------------------- Using built-in animations: Popular example (Login bear) --------------------

Go to: Marketplace Examples
Search for: Login bear (Animated Login Character)
Download the .riv file (Do not open to edit, it requires a paid subscription to export. Just download the file)
add the .riv file to your project (ie assets/rive_animation_assets/login_bear.riv)
in pubspec.yaml:
    assets:
      - assets/rive_animation_assets/
