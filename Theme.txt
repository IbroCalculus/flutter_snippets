
CHECK ALS0: Custom theme.txt

=============== Read theme values anywhere using: ============
Theme.of(context)

E.g:

Text(
  "Hello",
  style: TextStyle(
    color: Theme.of(context).colorScheme.primary,
  ),
)

------- More on using Theme.of(context) / colorScheme.primary below, after Setting Theme -----------



================ Setting Theme ===============
NB: theme, darkTheme, and themeMode

  @override
  Widget build(BuildContext context) {
    
    var theme = Theme.of(context);        // Create Theme.of(context) as a variable for easier usage.

    return MaterialApp(
      debugShowCheckedModeBanner: false,
      title: 'Theme Practice',
      themeMode: ThemeMode.light,

      theme: ThemeData(
        colorScheme: .fromSeed(
          seedColor: Colors.deepPurple,
          brightness: Brightness.light,
        ),
        textTheme: TextTheme(
          titleMedium: TextStyle(fontSize: 16, fontWeight: FontWeight.w500, fontStyle: FontStyle.italic, color: Colors.green),
          bodyMedium: TextStyle(fontSize: 14, fontWeight: FontWeight.w400),
        ),
        appBarTheme: AppBarTheme(
          // backgroundColor: Colors.lightBlue,                     // Without using theme. Not recommended
          backgroundColor: Theme.of(context).colorScheme.primary,   // Using theme. Recommended
          // backgroundColor: theme.colorScheme.primary,            // Optional above, variable.
          foregroundColor: Colors.white,
          centerTitle: true,
        ),
        fontFamily: 'Georgia',
        useMaterial3: true,
        textTheme: TextTheme().apply(
          bodyColor: Colors.red
        )
      ),

      darkTheme: ThemeData(
        colorScheme: ColorScheme.fromSeed(
          seedColor: Colors.deepPurple,
          brightness: Brightness.dark,
        ),

        appBarTheme: AppBarTheme(
          backgroundColor: Colors.black,
          foregroundColor: Colors.white,
          centerTitle: false,
        ),
        fontFamily: 'Roboto',
        useMaterial3: true,
      ),
      
      home: HomePage(),
    );
  }


======================= Using Theme.of(context) / colorScheme.primary =====================

------------------ How colorScheme: ColorScheme.fromSeed(seedColor: Colors.deepPurple), works -------------
It creates a color pallette from the Colors.deepPurple using material3 color algorithm. 
It generates color variants, ie; Primary, secondary, onSecondary, etc. Check google Docs; Flutter Reference - Theme

Use Material Theme Builder for your personal generation (Also available as a Figma plugin)

 ------------------ 1. ColorScheme Theme -----------------------
Check above for colorScheme defined above via: colorScheme: .fromSeed( ...

Usage:

Text(
  "Hello",
  style: TextStyle(
    color: Theme.of(context).colorScheme.primary,	// .primary, .secondary, surface etc
  ),
)


 ------------------ 2. TextTheme -----------------------
Check google Docs; Flutter Reference - Theme
NB: TextTheme is used for styling, including colors of a group of text 

Check above for textTheme defined via: textTheme: TextTheme(...
and note that they are default styles for each of the values (ie; .headlineLarge, etc) even if not custom-defined

Text(
  'Hello World',
  style: Theme.of(context).textTheme.headlineLarge,
)


 ------------------ 3. Component Theme -----------------------
ie; AppBarTheme, elevatedButtonTheme, bottomSheetTheme, etc

E.g:
elevatedButtonTheme: ElevatedButtonThemeData(
          style: ElevatedButton.styleFrom(

            shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(0)),
